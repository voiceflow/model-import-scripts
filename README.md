# Voiceflow NLU Model Import

Pass a supported model, get a [Voiceflow NLU CSV](https://learn.voiceflow.com/hc/en-us/articles/10587264500877) out.

## Usage

1. Clone

2. Install

    `npm i`

3. Build

    `npm run build`

4. Execute

    `node build/bin.js -m <MODEL> -i <INPUT FILE> -o <OPTIONAL OUTPUT FILE>`

## Models

1. `dialogflow-cx`
2. `einstein`
3. `lex-v1`
4. `nuance-mix`
5. `rasa2`
6. `watson`
